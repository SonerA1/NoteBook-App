# NoteBook-App
NoteBook-App

In the application, we can add new notes, update the saved notes, delete operations, and search notes in the search bar.

## Project Features
-MVVM

-Coroutines

-LifeCycle

-LiveData

-Room Database

-ViewModel

-Navigation
